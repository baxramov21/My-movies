# My-movies
Another gift for movie lovers
This app shows 2 types of movies: 
1 - newest premieres 
2 - the most high ranked films of all time
Also users can leave comments and read comments of other users. Like movies so they will be saved under favorites folder. Shorter: New generation of Netflix üòÅ.
Tech stack: Android SDK, Java, Retrofit2, Gson, RxJava, MVVM, Picasso, XML and etc.
